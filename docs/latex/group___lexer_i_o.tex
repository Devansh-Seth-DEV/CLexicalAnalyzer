\doxysubsubsection{Detailed Description}
File chunk handling utilities for lexical analysis. 

Provides structures and functions for\+:
\begin{DoxyItemize}
\item Reading source files into manageable chunks
\item Writing processed tokens back to files
\item Memory-\/managed file operations
\end{DoxyItemize}

\begin{DoxyNote}{Note}
All functions maintain binary-\/safe operations

Constructs a new file chunk object. 

This function dynamically allocates memory for a new {\ttfamily \doxylink{structfchnk__t}{fchnk\+\_\+t}} structure, initializing it to hold a chunk of data from a file. The {\ttfamily buff} pointer in the newly allocated {\ttfamily \doxylink{structfchnk__t}{fchnk\+\_\+t}} structure is not initialized (it may be {\ttfamily NULL} initially). The caller should ensure to allocate memory for {\ttfamily buff} and set {\ttfamily chksz} appropriately after the constructor is called.
