\doxysubsection{Detailed Description}
Complete token handling system for source code analysis. 

This header provides the complete infrastructure for\+:
\begin{DoxyItemize}
\item Token type classification (\#\+Token\+Classification)
\item Individual token representation (\#\+Token\+Structure)
\item Token collection management (\#\+Token\+Set)
\end{DoxyItemize}

\begin{DoxyNote}{Note}
The groups form a processing pipeline\+:
\begin{DoxyEnumerate}
\item Classification categorizes lexemes
\item Structures store the results
\item Collections manage token sequences 
\end{DoxyEnumerate}
\end{DoxyNote}
