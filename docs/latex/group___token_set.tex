\doxysubsection{Detailed Description}
Collection of tokens and operations. 

This group contains the token set structure (\doxylink{structtokset__t}{tokset\+\_\+t}) and all functions for managing collections of tokens, including\+:
\begin{DoxyItemize}
\item Set creation and destruction
\item Token counting and analysis
\item Set visualization and I/O 
\end{DoxyItemize}
